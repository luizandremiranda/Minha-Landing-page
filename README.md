## 📌 Sobre o projeto

Este projeto consiste em uma **landing page de portfólio profissional**, criada para apresentar minha trajetória, habilidades e evolução na área de desenvolvimento Front-End.

A página representa minha transição de carreira da indústria para tecnologia, destacando minha experiência profissional, formação em programação e meus objetivos como desenvolvedor.

O objetivo é criar uma presença digital profissional para apresentar projetos, conhecimentos técnicos e facilitar conexões com empresas e profissionais da área.

---

## 🎯 Objetivos

- Criar uma apresentação profissional online;
- Demonstrar conhecimentos em desenvolvimento Front-End;
- Apresentar minha trajetória profissional;
- Construir uma base para meu portfólio de projetos;
- Aplicar boas práticas de HTML, CSS e JavaScript.

3. Abra o arquivo:

```
index.html
```

ou utilize uma extensão como **Live Server** no VS Code.
